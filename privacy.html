Privacy Policy - Pablo Bot

This Privacy Policy explains how Pablo Bot handles user data:

  • We do not collect or store any personal data such as email, IP address, or DMs.
  • The bot may store:
    • Server configuration (e.g. welcome channel, game scores)
    • User IDs and command usage (only for internal use)
  • No data is shared with third parties.
  • All data is stored securely and is only used to enable bot features.
  • Users may contact the developer to request data deletion (e.g. score resets).

We comply with Discord's Developer Terms and community guidelines.
